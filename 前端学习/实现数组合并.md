### 实现多个数组合并，不去重

```javascript

1)不影响原数组

var a = [1, 3, 2];
var a = [4, 3, 5];
console.log([...a, ...b]);
=>
[1, 3, 2, 4, 3, 5]

var a = [1, 3, 2];
var a = [4, 3, 5];
console.log(a.concat(b));
=>
[1, 3, 2, 4, 3, 5]

var a = [1, 3, 2];
var b = [4, 3, 5];
console.log([].concat.apply([], [a, b]))
=>
[1, 3, 2, 4, 3, 5]

2)在原数组的基础上

var a = [1, 3, 2];
var b = [4, 3, 5];
[].push.apply(a, b);
console.log(a);
=>
[1, 3, 2, 4, 3, 5]

```

### 实现多个数组合并，去重


```javascript

1)利用new Set([iterable])可以创建没有重复值的集合，再利用Array.from(arrayLike[, mapFn[, thisArg]])将Set转为真实数组

var a = [1, 3, 2];
var b = [4, 3, 5];
function combine() {
  let arr = [].concat.apply([], arguments);
  return Array.from(new Set(arr));
}
console.log(combine(a, b))
=>
[1, 3, 2, 4, 5]

```
